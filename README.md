# Голосовой бот (Android)

Голосовой агент для Android с русским распознаванием речи, управлением
устройством по голосу и диктовкой текста с исправлением грамматики через Claude.

## Что умеет

- **Гибридное распознавание речи**: офлайн (Vosk, русская модель) для команд +
  Claude API для исправления надиктованного текста.
- **Полное управление устройством** через Accessibility Service: навигация
  (домой/назад/последние), прокрутка, нажатия по тексту на экране, запуск
  приложений, громкость.
- **Диктовка**: надиктованный текст вставляется в активное поле ввода
  (мессенджеры, заметки и т.п.), при этом Claude исправляет ошибки, пунктуацию
  и грамматику.
- **Шаблоны текста**: сохраняете часто используемые фразы и вставляете их
  голосом — «шаблон приветствие».
- **Главный выключатель** on/off + постоянное уведомление, пока бот слушает.

## Голосовые команды

| Скажите | Действие |
|---|---|
| `домой` | На главный экран |
| `назад` | Кнопка «Назад» |
| `последние` | Недавние приложения |
| `шторка` | Открыть уведомления |
| `прокрути вниз` / `вверх` | Прокрутка экрана |
| `громче` / `тише` | Громкость |
| `открой <название>` | Запустить приложение по названию |
| `нажми <текст>` | Нажать элемент с этим текстом на экране |
| `шаблон <имя>` | Вставить сохранённый шаблон |
| `печатай …` | Войти в режим диктовки |
| `стоп` | Завершить диктовку |

Ключевые слова легко изменить в файле
[`CommandRouter.kt`](app/src/main/java/com/voicebot/app/command/CommandRouter.kt).

## Как собрать APK

Есть два пути. Выбирайте любой.

### Вариант A — автосборка в облаке (без установки Android Studio) — рекомендуется

1. Создайте репозиторий на GitHub и загрузите туда содержимое папки `bot`.
2. Откройте вкладку **Actions** — workflow «Build APK» запустится автоматически
   (или нажмите **Run workflow**).
3. Через 3–5 минут откройте завершённый запуск → раздел **Artifacts** →
   скачайте `voicebot-debug-apk`. Внутри — готовый `app-debug.apk`.
4. Перекиньте APK на телефон и установите (разрешив «установку из неизвестных
   источников»).

Скрипт сборки уже настроен: [`.github/workflows/build-apk.yml`](.github/workflows/build-apk.yml).

### Вариант B — локально в Android Studio

1. Установите [Android Studio](https://developer.android.com/studio).
2. **File → Open** → выберите папку `D:\bot`.
3. Дождитесь синхронизации Gradle (Android Studio сам подтянет нужный Gradle;
   при запросе разрешите сгенерировать Gradle Wrapper).
4. **Build → Build App Bundle(s) / APK(s) → Build APK(s)**.
5. APK появится в `app/build/outputs/apk/debug/`.

> Если собираете из командной строки без Android Studio, один раз выполните
> `gradle wrapper` (нужен установленный Gradle), затем `./gradlew assembleDebug`.

## Настройка после установки

1. Откройте приложение **Голосовой бот**.
2. Вставьте **Claude API ключ** (получить: <https://console.anthropic.com>) и
   нажмите «Сохранить ключ». Без ключа диктовка работает, но без исправления
   грамматики (выключите тумблер «Исправлять грамматику»).
3. Нажмите **«Открыть спец. возможности»** и включите службу
   «Голосовой бот — управление устройством». Это даёт полный контроль над
   экраном (нажатия, прокрутка, ввод текста).
4. Включите главный тумблер **«Бот включён»**. При первом запуске бот скачает
   русскую модель распознавания (~45 МБ) — нужен интернет только один раз.
5. Разрешите доступ к микрофону и уведомлениям.

Теперь можно говорить команды. Бот слушает в фоне, пока горит уведомление.

## Структура проекта

```
bot/
├─ app/src/main/
│  ├─ java/com/voicebot/app/
│  │  ├─ ui/MainActivity.kt              # панель управления, тумблер on/off
│  │  ├─ service/VoiceService.kt         # фоновый сервис + микрофон
│  │  ├─ service/BotAccessibilityService.kt  # действия на экране
│  │  ├─ recognition/VoskRecognizer.kt   # офлайн-распознавание
│  │  ├─ recognition/ModelManager.kt     # загрузка модели
│  │  ├─ command/CommandRouter.kt        # команды → действия (триггеры здесь)
│  │  ├─ llm/ClaudeClient.kt             # исправление текста через Claude
│  │  └─ data/Prefs.kt                   # настройки, ключ, шаблоны
│  ├─ res/                               # интерфейс, иконки, строки
│  └─ AndroidManifest.xml
├─ .github/workflows/build-apk.yml       # автосборка APK
└─ build.gradle.kts / settings.gradle.kts
```

## Технические детали

- `minSdk 26` (Android 8.0+), `targetSdk 34`.
- Распознавание: [Vosk](https://alphacephei.com/vosk/), модель
  `vosk-model-small-ru-0.22`.
- Грамматика: модель `claude-haiku-4-5` (быстрая и дешёвая; можно поменять в
  `ClaudeClient.kt`).

## Важно про разрешения

Accessibility Service даёт приложению широкий доступ к экрану — это необходимо
для управления устройством голосом. Приложение работает локально, ключ API
хранится только на устройстве. Устанавливайте и включайте службу осознанно.
